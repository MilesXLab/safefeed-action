# Netherlands (NVWA) Infant Formula Recall - February 2026

## Source Information
- **Official Recall Page**: https://www.nutriciavoorjou.nl/terugroepactie.html
- **Brand**: Nutrilon (Danone Netherlands)
- **Date**: February 8, 2026
- **Total Batches**: 44 batches across 14 product varieties
- **Identifier**: THT (Tenminste Houdbaar Tot / Best Before Date)

## Key Information
- This is a **NEW REGION** not previously in the database
- Netherlands uses THT dates as primary batch identifiers (not traditional batch codes)
- Recall covers Nutrilon 1, 2, Profutura Duobalans, AR, and Pepti varieties
